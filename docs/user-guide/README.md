This track is for operators and team leads who run OpsClaw in real environments.

- Installation and first run (`opsclaw init` path)
- Platform setup (Slack, later Discord and Telegram)
- Day-2 operations (approvals, runbooks, troubleshooting)
- Skills and SOUL customization for production squads
